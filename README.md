# SCM Personnel Management System

نظام إدارة الموظفين للسلسلة اللوجستية (Supply Chain Management Personnel Management)

## الوصف

تطبيق ويب متكامل لإدارة الموظفين والعاملين في نظام السلسلة اللوجستية. يوفر التطبيق واجهة رسمية ديناميكية مع دعم اللغة العربية واتجاه RTL (من اليمين إلى اليسار).

## الميزات الرئيسية

- ✅ **نظام تسجيل الدخول** (Login System)
- 📊 **لوحة معلومات** (Dashboard) تعرض:
  - عدد الموظفين في الطريق (On Route)
  - الموظفين في الانتظار (Idle)
  - الموظفين غير في الخدمة (Off-Duty)
- 🔍 **البحث والتصفية** (Search & Filter)
- 📱 **تصميم متجاوب** (Responsive Design)
- 🌐 **دعم RTL** (دعم اللغة العربية كاملة)
- 🎨 **واجهة مستخدم محترفة** مع ألوان براقة

## التكنولوجيا المستخدمة

- **HTML5** - لغة الترميز
- **CSS3** - التصميم والأنماط
- **JavaScript (Vanilla)** - البرمجة الديناميكية
- **Font Awesome** - أيقونات احترافية

## كيفية الاستخدام

1. افتح ملف `Test.html` في متصفح الويب
2. سيظهر شاشة الترحيب (Splash Screen) لمدة 3 ثوانٍ
3. أدخل بيانات المستخدم للدخول
4. يمكنك البحث والتصفية بين الموظفين

## مميزات الواجهة

- **Splash Screen**: شاشة ترحيب احترافية
- **Login Screen**: نموذج تسجيل دخول آمن
- **Metrics Dashboard**: لوحة معلومات حية
- **Personnel List**: قائمة الموظفين مع تفاصيلهم
- **Navigation Bar**: شريط تنقل سفلي
